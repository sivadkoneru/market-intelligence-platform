# Codex Notes

This directory holds repo-local Codex helper material. It is intentionally small so the authoritative project rules stay in root `AGENTS.md` and `CLAUDE.md`.

Use these files when working in this repository:

- `../AGENTS.md` - fast agent contract and verification commands.
- `../CLAUDE.md` - full architecture, conventions, ports, and completed-task state.
- `checklists/change.md` - quick checklist before and after code changes.

No secrets, generated artifacts, or personal environment state belong here.
